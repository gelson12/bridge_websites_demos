# bridge_websites_demos

Demo websites auto-generated by the Bridge Company pipeline
(see `gelson12/super-agent` → `n8n/bridge_website_builder.json`).

Each prospect's demo lives at `/{slug}/index.html` and is served by GitHub Pages at
https://gelson12.github.io/bridge_websites_demos/{slug}/

Demos are auto-deleted after 12 hours unless the prospect accepts the offer.
The `_template/` folder is the base template injected by the builder workflow.

Do not hand-edit this repo — all content is machine-generated.
